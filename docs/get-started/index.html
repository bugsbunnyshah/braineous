Step 1

Install MongoDB

brew tap mongodb/brew
brew update
brew install mongodb-community@7.0

Step 2

Download Braineous

Braineous-1.0.0-CR3

More instructions

Release

Step 3

Start Zookeeper

cd braineous-1.0.0-cr3/bin
./start_zookeper.sh

Step 4

Start Kafka

cd braineous-1.0.0-cr3/bin
./start_kafka.sh

Step 5

Start Flink

cd braineous-1.0.0-cr3/bin
./start_flink.sh

Step 6

Start Hive Metastore

cd braineous-1.0.0-cr3/bin
./start_metastore.sh

Step 7

Start Braineous

cd braineous-1.0.0-cr3/bin
./start_braineous.sh

Step 8

Test Braineous Installation

cd braineous-1.0.0-cr3/bin
./test_installation.sh
